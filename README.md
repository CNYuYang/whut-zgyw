# 武汉理工大学中国语文刷时长的小工具



该项目是采用node.js所写的小工具。可自动帮你刷中国语文网站的在线时长。

中国语文网站：<http://59.69.102.9/zgyw/index.aspx>

## 环境要求



- node.js



## 相关依赖



- cheerio

- request




## 使用说明

获取本项目：

```bash
git clone https://github.com/Reyunn/whut-zgyw.git
```

安装依赖：

```bash
npm install
```

运行：

```javascript
node index.js 学号 密码
//如 node index.js 0121709360222 0121709360222
```
